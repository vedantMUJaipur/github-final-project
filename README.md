# Simple Interest Calculator

A simple Bash script application that calculates simple interest based on user inputs for principal, rate of interest, and time period.

## How It Works
The script prompts the user to input the mathematical variables, applies the classic financial computing formula, and outputs the final calculated integer total.

## Mathematical Formula
Simple Interest = (P * R * T) / 100

## Example Usage
```bash
Enter the principal:
1000
Enter rate of interest per year:
5
Enter time period in years:
2
The simple interest is:
100
